Why Context Pilot?

            
Every token matters. Context Pilot gives you surgical control over your AI's context window.

            
            

                

                    
üéØ

                    
Context Control

                    
Open files, git diffs, grep results, and more as context panels. See exactly what your AI sees ‚Äî token counts and all. Close what you don't need.

                

                

                    
‚ö°

                    
Blazing Fast

                    
Built in Rust with ratatui. Sub-millisecond rendering. Instant file operations. No Electron, no web browser, no bloat ‚Äî just raw terminal performance.

                

                

                    
üîå

                    
Modular Architecture

                    
Activate only the modules you need. Git, GitHub, files, tree, console, todos, memory, scratchpad ‚Äî each module adds its own tools and panels.

                

                

                    
ü§ñ

                    
Multi-Provider

                    
Works with Anthropic Claude, OpenAI, DeepSeek, Grok (xAI), and Groq. Switch providers and models mid-conversation with a single keystroke.

                

                

                    
üß†

                    
Persistent Memory

                    
Memories and todos persist across conversations. The AI remembers your preferences, project conventions, and ongoing tasks.

                

                

                    
üîÑ

                    
Auto-Continuation

                    
The Spine module handles stream control with configurable guard rails ‚Äî max tokens, cost limits, and automatic continuation for long tasks.

                

            

        

    


    
    

        

            
Modular by Design

            
Activate what you need. Deactivate what you don't. Each module is independent.
